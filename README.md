# Prédiction du risque d'AVC par Machine Learning

## Objectif

Développer et comparer plusieurs modèles de Machine Learning afin de prédire le risque de survenue d'un AVC à partir de données médicales.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git
- GitHub

## Modèles étudiés

- Régression Logistique
- Arbre de Décision
- Random Forest

## Évaluation

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- Matrice de confusion
